# Pensando-en-Ciencia-de-Datos
Libro Interactivo de Ciencia de Datos


## Este repositorio contiene una guía práctica de ciencia de datos, desarrollada como parte de mis prácticas profesionales en la Facultad de Ciencias Físico Matemáticas, Puebla, México. Está organizada en 17 capítulos, cada uno con explicaciones teóricas, ejemplos de código reproducibles y aplicaciones prácticas usando datasets reales.
