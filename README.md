# Guía de Ciencia de Datos – 17 Capítulos Interactivos
Libro Interactivo de Ciencia de Datos

Este repositorio contiene una guía práctica de ciencia de datos, desarrollada como parte de mis prácticas profesionales en la Facultad de Ciencias Físico Matemáticas. Está organizada en 17 capítulos, cada uno con explicaciones teóricas, ejemplos de código reproducibles y aplicaciones prácticas usando datasets reales.

### Contenido
- Introducción a la ciencia de datos
- Limpieza y preprocesamiento de datos
- Análisis exploratorio y visualización
- Estadística descriptiva y pruebas básicas
- Algoritmos de machine learning: regresión, clasificación (SVM, KNN, árboles, etc.)
- Clustering y segmentación
- Evaluación de modelos y métricas
- Automatización de flujos de trabajo y documentación reproducible
- Casos de estudio con datasets reales
Cada capítulo está documentado en Jupyter Notebooks, combinando teoría, código y visualizaciones interactivas.

### Tecnologías y Herramientas
- Python (pandas, numpy, matplotlib, seaborn, scikit-learn)
- R para análisis complementario
- Jupyter Notebooks
- Git/GitHub para control de versiones
- Visual Code
